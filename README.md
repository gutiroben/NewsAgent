# NewAgent

NewAgent는 다양한 AI 관련 뉴스 피드들을 매일 수집하여 LLM을 이용해 종합 뉴스로 만들어 메일로 보내주는 서비스입니다.

## 주요 기능

- **뉴스 수집**: 다양한 AI 관련 뉴스 피드에서 최신 정보를 자동으로 수집
- **LLM 기반 종합**: 수집된 뉴스를 LLM을 활용하여 요약 및 종합
- **자동 메일 발송**: 매일 종합된 뉴스를 이메일로 자동 발송
